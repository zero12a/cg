* HTML OBJ
코드검색(폼/컨디션) : 코드=GRPID-COLID, 이름=GRPID-COLID-NM
코드검색(그리드) : 3개 코드^이름^GRPID
링크뷰 : 2개 링크=GRPID-COLID-LINK, 이름=GRPID-COLID-NM, 숨김객체=DIV-GRPID-COLID
라디오 : 홀더 GRPID-COLID-HOLDER
체크박스 : 홀더 GRPID-COLID-HOLDER
이미지뷰어 : 홀더 GRPID-COLID-HOLDER
파일 : 4개 파일객체=GRPID-COLID, 링크=GRPID-COLID-LINK, 파일이름=GRPID-COLID-NM, 숨김객체=DIV-GRPID-COLID

* 조회 
코드검색(폼/컨디션) : 2개 코드^이름
코드검색(그리드) : 3개 코드^이름^GRPID
링크뷰 : 2개 링크^이름
라디오 : 1개 
체크박스 : N개  (형식:값1,값2,값3....)
이미지뷰어 : N개 (형식:값1, 값2, 값3, 값의 분해 : 상세링크^이미지경로)
파일 : 2개 링크^파일이름
